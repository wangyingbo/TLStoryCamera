##介绍
>仿照instagramStory & WeiboStory。可以在拍摄视频后贴图，贴字，涂鸦。

##特性
>1·基本实现WeiboStory的大部分功能
>
>2·支持码率，美颜开关等多项配置
>
>3·支持长按录像&短按拍照
>
>4·使用Swift 3开发
>

##依赖框架
>GPUImage
>

##使用说明
>克隆项目后，需要使用Cocoapods updte/install
>
>部分图片素材来源于Weibo，如需将TLStoryCamera用于实际项目，请替换资源图片
>
>目前项目尚处于开发阶段，不建议直接在企业项目中使用。
